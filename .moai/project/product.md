# MIRIP 제품 문서

> 미립(MIRIP) - 미술 입시의 모의고사 + 성적표

---

## 1. 서비스 정의

| 항목 | 내용 |
|------|------|
| **플랫폼명** | 미립 (MIRIP) |
| **컨셉** | 미술 입시의 모의고사 + 성적표 |
| **핵심 가치** | "내 그림이 목표 대학 기준 어느 위치인가"를 객관적 데이터로 확인 |
| **타겟** | 미술 입시생 (연간 5만 명), 미술학원/화방/지자체 |

### 서비스 한 줄 설명

> "당신의 작품, 어디까지 갈 수 있을까요?" - 대학별 합격 데이터를 학습한 AI가 객관적으로 진단합니다.

---

## 2. Pain Point - Solution 매핑

| # | Pain Point | Solution | 플랫폼 기능 |
|---|------------|----------|-------------|
| 1 | 평가의 주관성 | AI 합격 예측 | **미립에듀** (AI 진단) |
| 2 | 정보 파편화 | Kaggle형 공모전 플랫폼 | **미립콤프** (공모전) |
| 3 | 경력 비표준화 | 예술가용 LinkedIn | **크레덴셜** (포트폴리오) |

### 기존 문제점

- 월 80~150만원의 학원비, 그러나 기준 없는 평가
- "지금 내 실력이 어느 정도인지" 알 수 없음
- 수상 이력과 포트폴리오가 흩어지고 사라짐

### MIRIP의 해결책

| 문제 | MIRIP 솔루션 |
|------|-------------|
| 불명확한 평가 기준 | AI 기반 객관적 진단 및 점수화 |
| 실력 파악 어려움 | 대학별 합격 예측 데이터 제공 |
| 이력 관리 부재 | 통합 크레덴셜 시스템 |
| 공모전 정보 분산 | 원스톱 공모전 플랫폼 |

## 3. Customer Decision Journey (CDJ)

```
+---------------------------------------------------------------------+
|  인지 (Awareness)     ->   유입 (Acquisition)   ->   전환 (Conversion)  |
|  공모전 정보 큐레이션      콘테스트 참가            AI 진단 체험->구독    |
|  (무료)                   (무료/유료)             (건당/월정액)        |
+---------------------------------------------------------------------+
                                    |
                                    v
+---------------------------------------------------------------------+
|  리텐션 (Retention)                                                  |
|  프로필(크레덴셜) 축적 -> 이탈 비용 증가 -> Lock-in                      |
+---------------------------------------------------------------------+
```

### 4단계 연결 구조

```
공모전 -> 크레덴셜 -> AI 진단 -> 커리어
  (참여)    (축적)     (분석)    (확장)
```

## 4. 타겟 사용자

| 유형 | 설명 | 핵심 니즈 |
|------|------|-----------|
| **입시생** | 미대 입시 준비 중인 학생 | 실력 진단, 합격 가능성 예측 |
| **학부모** | 입시생 자녀를 둔 부모 | 객관적 평가, 비용 효율 |
| **신진 작가** | 데뷔 준비 중인 예술가 | 포트폴리오, 공모전 정보 |
| **주최자** | 공모전/전시 운영자 | 심사 효율화, 작품 관리 |

### 사용자별 상세 시나리오

#### 입시생 시나리오

1. 플랫폼에 작품 이미지 업로드
2. AI 진단을 통해 대학별 합격 가능성 확인
3. 피드백을 바탕으로 보완점 파악 및 연습
4. 공모전 참여를 통해 실전 경험 축적
5. 수상 이력이 크레덴셜로 자동 기록

#### 신진 작가 시나리오

1. 플랫폼에 포트폴리오 등록
2. 관련 공모전 정보 알림 수신
3. 공모전 참여 및 수상
4. 크레덴셜 프로필 강화
5. 갤러리/기관 연결 및 커리어 확장

## 5. 핵심 기능

### 5.1 AI 합격 예측 진단 (미립에듀)

- 대학별 합격 데이터를 학습한 AI 모델 기반
- 작품 이미지 분석을 통한 객관적 진단
- 대학별 합격 확률 점수 제공 (예: 서울대 74, 홍익대 81, 국민대 69)
- 4가지 평가 축: 구성력, 명암/질감, 조형완성도, 주제해석력
- 강점 및 보완점 피드백 제공

### 5.2 공모전 플랫폼 (미립콤프)

- 미술 공모전 정보 통합 제공
- 참여 및 접수 관리
- 공모전 개최 기능 (주최자용)
- 수상 이력 자동 연동
- 분야별 필터: 시디/산디/회화/공예

### 5.3 크레덴셜 관리

- 수상 이력 및 포트폴리오 통합 관리
- 경력 증명서 발급
- 프로필 기반 커리어 확장
- 채용 및 거래 연결
- 자동 이력 적재 (공모전 수상, AI 진단 결과)

## 6. 가치 제안

### 경쟁 우위

| 요소 | 기존 서비스 | MIRIP |
|------|------------|-------|
| 평가 방식 | 주관적 피드백 | AI 기반 객관적 점수 |
| 데이터 | 없음 | 대학별 합격 데이터 |
| 이력 관리 | 분산/수동 | 통합 자동화 |
| 공모전 정보 | 분산 | 원스톱 플랫폼 |

### 핵심 차별점

1. **객관적 진단**: 대학별 합격 데이터 기반 AI 평가
2. **통합 플랫폼**: 공모전 + 진단 + 크레덴셜 원스톱
3. **자동 이력 관리**: 활동 기록의 자동 축적
4. **Lock-in 효과**: 축적된 이력으로 인한 이탈 비용

## 7. 디자인 철학

MIRIP은 **미술관/갤러리 컨셉**의 디자인을 추구합니다.

### 스타일 키워드

- 미니멀 & 여백 중심 (White space 활용)
- 갤러리처럼 작품이 돋보이는 레이아웃
- 세리프 폰트로 고급스러움
- 얇은 라인, 섬세한 디테일
- 플랫하고 클린한 UI

### 디자인 레퍼런스

- 국립현대미술관, 리움미술관 웹사이트
- 고급 갤러리 초대장 느낌
- "조용하지만 강렬한" 첫인상
