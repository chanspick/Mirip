# SPEC-COMP-001 인수 조건

## 공모전 목록 페이지

- [ ] `/competitions` 접근 시 공모전 카드 그리드 표시
- [ ] 카드에 썸네일, 제목, D-Day, 상금, 분야 표시
- [ ] 분야 필터 동작 (시각디자인, 산업디자인, 공예, 회화)
- [ ] 상태 필터 동작 (진행중, 마감임박, 종료)
- [ ] 정렬 옵션 동작 (마감순, 상금순, 인기순, 최신순)
- [ ] 무한 스크롤 또는 페이지네이션 동작
- [ ] 로딩 스켈레톤 UI 표시

## 공모전 상세 페이지

- [ ] `/competitions/:id` 접근 시 상세 정보 표시
- [ ] 헤더에 썸네일, 제목, D-Day 배지, 참가자 수, 상금 표시
- [ ] 탭 메뉴 동작 (상세정보, 출품작, 결과)
- [ ] URL 해시로 탭 상태 유지
- [ ] 존재하지 않는 ID 접근 시 404 표시
- [ ] "출품하기" 버튼 클릭 시 출품 페이지로 이동

## 출품 페이지

- [ ] 이미지 드래그앤드롭 업로드 동작
- [ ] 이미지 미리보기 표시
- [ ] 업로드 진행률 표시
- [ ] 작품 정보 입력 폼 동작
- [ ] 필수 필드 검증 동작
- [ ] 제출 전 미리보기 표시
- [ ] 제출 완료 시 확인 모달 표시
